YouTube Live Chat Blocker Options

Import / Export Rules

You can export your current rule list as a JSON file, or import an existing JSON file to restore or migrate your rules.

Help & Usage

How to use

  • Click the extension icon to open the popup window.
  • Add rules to block chat messages. Messages matching active rules will be hidden.
  • You can toggle individual rules or turn off the blocker completely using the top switch.

Regular Expression Examples

If you choose "Regular expression" as the rule type, you can use patterns to block messages. Here are some useful examples:

Pattern Meaning Example Matches
word$ Matches messages ending with 'word' hello word, this is a word
^word Matches messages starting with 'word' word is good, word
^a$ Exactly the specific character 'a' a
apple|orange Contains either 'apple' or 'orange' I like apple, orange juice
w{3,} Contains 3 or more consecutive 'w's www, wwww, hello wwww
